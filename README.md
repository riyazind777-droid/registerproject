# 📝 Registration Form Validation

A responsive **Registration Form** built using **HTML, CSS, and JavaScript** that performs real-time form validation. The application validates user input before allowing form submission, ensuring that all required fields are correctly filled.

## 🚀 Features

- ✅ Username validation (required field)
- ✅ Email format validation
- ✅ Password validation (minimum 6 characters)
- ✅ Real-time error messages
- ✅ Dynamic input border highlighting
- ✅ Submit button validation
- ✅ Simple and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🎯 Validation Rules

### Username
- Cannot be empty

### Email
- Must be in a valid email format

Example:
eamil.com X
eamil@gmail.com  correct

### Password
- Minimum 6 characters
